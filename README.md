# Snake Game made in Vanilla JS
## How To Play
* Clone this Repo.
```
git clone https://github.com/bmviniciuss/SnakeJs.git
cd SnakeJs
```
* Then type in a terminal on the project folder.
```
python3 -m http.server
```
* Open the localhost in port 8000
